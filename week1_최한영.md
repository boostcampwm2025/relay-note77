# 퀘스트 4. 슬랙 질문 요약 정리봇

선택 이유: 동료의 질문을 통해 내가 놓쳤던 부분을 체크하고 학습할 수 있을 것으로 기대한다.

질문은 자체적으로 가공해서 사용한다.

Prompt 형식

Question. 
1. prompt. 
질문을 보고 답변 형식에 맞게 미션 분석, 토론 요약 결과, 토론의 요약 결과와 너의 분석을 비교하여 각 항목마다 100자 이내로 작성해
2. 질문
{   
    미션: '',
    질문자: '',
    사용자 질문: '',
    토론 내용:
    {
        참여자1: '',
        참여자2: '',
    }
}

3. 답변 형식
{ 
    AI의 문제 분석: '',
    토론 요약 결과:'',
    비교 분석:'',
}
